# A-Chinese-Essay-Reading-Helper
这个项目是一个PDF文献阅读器（只能读PDF！）。由于作者对市面上的文献阅读器的前端感觉到十分的……朴素，故而尝试设计了这一款行业前沿的关注器。

计划包含功能如下：
## 文献翻译
最基础的功能。
## 文献提炼 
其中主要包括两个方面：
**1.** 单一文献的重要内容提炼
**2.** 多个文献的核心内容与发展方向提炼。

## 文献小卡命题
考虑LoRA微调
这个功能的Motivation来自于作者的期末复习。作者发现，把资料丢给LLM，让他命制习题会极大提升复习效率，于是引入该功能。
LLM在阅读了文献后，使用该功能会让LLM命制一些题目，这些题目关于这篇文献，可以提出很多问题，就像我们高中看课本之后，会有例题帮我们掌握概念一样。


## 一键文件整理（慎用）
一键文件整理功能会根据上面的文献提炼功能提炼出来的发展路线，按照时间，题目整理文献，但不保证结果。
